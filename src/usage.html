Usage

This page contains information how to use PokerTimer. It assumes you are familiar with Poker terms and tournament rules.

The Screen

PokerTimer was designed to fit on your screen all at once. This is true for desktops, but for tablets and mobile phones, you might need to scroll down to see everything.

In desktop mode the center contains the most important information about the tournament. The left hand side contains various parameters and the right hand side has settings. When going to tablet the settings will first move to the bottom. When going mobile, the parameters go below the main controls. The timer is always at the center/top.

Starting and stopping

Right under the main timer you will see the tournament control buttons.

Start
This will start the clock if it is stopped or paused
Stop
This will pause the clock if it is started
Reset tournament
This will reset the tournament. All progress (levels, buyins, rebuys, etc) is lost. PokerTimer will ask for your confirmation before doing this.
Skip level
This will immediately skip a level (no confirmation, no going back). Can be used to skip ahead or if you forgot to start the timer to adjust for the missed time. Also if you accidently reset the tournament, you can go back to where you were.

Tournament participants

On the left hand side there are multiple buttons to control the participants. Here you can add or remove players and also mark rebuys and addons.

Player numbers, rebuys and addons do not affect the overall flow of the timer. They are used to show the amount of chips in play, and to tell you how to devide the prize money.

Settings

There are a lot of settings you can make in PokerTimer. PokerTimer uses presets to group together all settings. Settings such as buyin amount, buyin chips, levels and prize sharing is all put into presets. You can have multiple presets, and change between them.

All settings other than preset settings always affect the currently selected preset. To select a preset, press the Presets button and choose the preset you want to load.

Tips

Here are some tips and tricks how to use the application.

Internet Explorer 9: Pin to task bar

IE9+ allows you to pin application to the Windows task bar. When you are on the main application page of PokerTimer (NOT IN THIS USERS GUIDE!), drag the TAB in which the web page is displayed to the Windows task bar.

From now on, you can start the application directly from there.

Google Chrome: Pin to desktop/startmenu/taskbar

Google Chrome allows a similar pinning to take place. The method is slightly different.

Click the wrench icon (right side of address bar), select Tools, and then select Create application shortcuts.... Check the desired options.

Offline usage (Chrome, Firefox, Safari)

Poker Timer can be used offline, that is, without an internet connection. You need to visit the page once to do this - obviously you have done so by reading this. After this, you can just use the same address to access the application, without an internet connection.

You might want to bookmark the site or create application shortcust (see above) so you can access the application without an internet connection.

This works in:

  • Google Chrome
  • Mozilla Firefox (v7+)
  • Apple Safari (v4.0+)
  • iOS Safari (v3.2+)
