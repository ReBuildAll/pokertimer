Usage

Here are some tips and tricks how to use the application.

Internet Explorer 9: Pin to task bar

IE9+ allows you to pin application to the Windows task bar. When you are on the main application page of PokerTimer (NOT IN THIS USERS GUIDE!), drag the TAB in which the web page is displayed to the Windows task bar.

From now on, you can start the application directly from there.

Google Chrome: Pin to desktop/startmenu/taskbar

Google Chrome allows a similar pinning to take place. The method is slightly different.

Click the wrench icon (right side of address bar), select Tools, and then select Create application shortcuts.... Check the desired options.

Offline usage (Chrome, Firefox, Safari)

Poker Timer can be used offline, that is, without an internet connection. You need to visit the page once to do this - obviously you have done so by reading this. After this, you can just use the same address to access the application, without an internet connection.

You might want to bookmark the site or create application shortcust (see above) so you can access the application without an internet connection.

This works in:

  • Google Chrome
  • Mozilla Firefox (v7+)
  • Apple Safari (v4.0+)
  • iOS Safari (v3.2+)
